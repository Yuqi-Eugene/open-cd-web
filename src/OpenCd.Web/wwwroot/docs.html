目录结构

dataset_root/
  train/{A,B,label}
  val/{A,B,label}
  test/{A,B,label}

命名与配对

同名严格对齐，A/B/label 必须都存在，例如：000123.tif。

图像尺寸

建议裁块为 256x256 或 512x512；单次实验内保持一致。

存储格式

推荐 tif/tiff/png。训练前建议统一为 uint8。

nodata 约束

支持 nodata。建议预处理统一映射到 0 或 255，并在模型配置同步 ignore_index。

流程建议

  1. 批量命名校正。
  2. uint8 转换与 nodata 处理。
  3. 自动切分 train/val/test。
  4. 训练、验证、测试并查看指标曲线。
